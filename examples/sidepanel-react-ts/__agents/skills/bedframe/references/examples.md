# Bedframe Examples

Use local project files and the Bedframe examples repository to ground implementation decisions when conventions are unclear.

## Prioritize

- the current project's `AGENTS.md`
- the current project's nearest-folder `AGENTS.md`
- the current project's `src/_config/bedframe.config.ts`
- the current project's existing manifests, pages, and scripts

## Then compare against

- official Bedframe examples
- existing browser-specific manifest patterns
- existing page and runtime entrypoint shapes already used by the scaffold

Examples are reference material, not a license to bypass the project's current Bedframe contract.
